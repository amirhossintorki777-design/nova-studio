# Nova Studio v4

Final responsive portfolio website for Nova Studio.

## Run locally

1. Open the folder in VS Code.
2. Right-click `index.html`.
3. Choose **Open with Live Server**.

## Publish

After copying these files into the GitHub-connected project folder:

```bash
git add .
git commit -m "Publish Nova Studio v4"
git push
```

## Important

The contact form is visual only until connected to an email/form service.
